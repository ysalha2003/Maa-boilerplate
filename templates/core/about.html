About This Project

Welcome to the Django + Tailwind CSS Boilerplate! This project provides a modern, robust foundation for building web applications with Django.

Key Features

  • Modern Stack: Django 5.1+, Tailwind CSS 3.4+, and DaisyUI 4+
  • Custom User Model: Extensible user model ready from the start
  • Full Authentication: Complete styled flows for user management
  • Themeable UI: Light/Dark mode and theme switching built-in
  • Production Ready: Security and deployment best practices

Philosophy

We believe a good boilerplate should get out of your way and let you focus on what makes your application unique. This setup prioritizes best practices, security, and developer experience.
